Capture Flow

1. Trigger detected → brief camera flash animation (white overlay, 200ms fade)

2. Canvas state captured via canvas.toBlob()

3. Styled overlay composited onto capture:

• Score + combo count (top-left)

• Level number (top-right)

• Trigger label — "10X COMBO!" / "BOSS DEFEATED" / "FLAWLESS" (center, large)

• "NEON BREAKOUT" watermark (bottom, subtle)

4. Share button appears bottom-right for 5 seconds, then fades

Sharing

Mobile

Uses navigator.share() with the PNG blob. Opens native share sheet — iMessage, Instagram stories, Twitter, etc.

Desktop

Uses navigator.clipboard.write() to copy image to clipboard. Toast notification: "Screenshot copied!" Paste anywhere.

Cooldown: 10-second minimum between captures to avoid spam during intense sequences.

Gallery: Last 5 captures stored in memory (not localStorage — session only). Viewable from pause menu. Tap to re-share.

Non-intrusive: Flash is brief, share button is small and fades. Gameplay never pauses or interrupts.
