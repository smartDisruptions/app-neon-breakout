Design Review: Visual Spectacle (2 of 4)

Four features that make the game feel and sound alive

4. Procedural Synthwave Audio

Zero Audio Files — 100% Web Audio API

Soundtrack

A looping synthwave bassline built from OscillatorNodes + BiquadFilters. Tempo is steady (~120 BPM). The bass drives a global "pulse" value that modulates glow radius on all neon elements — everything breathes with the beat.

Reactive SFX

Brick hit: Pitched synth note based on brick row. Top row = high pitch, bottom = low. Destroying bricks creates emergent melodies as you play.

Combo builds: Reverb and distortion increase with combo count. At 10+ combo, the audio is thick and intense.

Paddle contact: Low-frequency thud. Satisfying impact feel.

Ball loss: Descending pitch sweep — the "falling" sound.

Power-up collect: Rising arpeggio, 3 quick notes.

Activation: Audio starts on first user interaction (click/tap to start game). Respects browser autoplay policies.

Mute toggle: 'M' key or a small speaker icon in the HUD corner.

5. Synthwave Horizon Background

Behind the Game Field

RENDERED ON OFFSCREEN CANVAS

Grid scrolls toward camera. Speed + sun pulse tied to combo.

Combo reactivity: At 0 combo, grid scrolls slowly, sun is dim. At 5+ combo, grid accelerates. At 10+, sun pulses, faint scanline ripples cross the horizon.

Performance: Pre-rendered to offscreen canvas, composited behind game layer. Updated every 2 frames, not every frame. Near-zero cost.

6. CRT Post-Processing Toggle

Retro Arcade Mode

Effects Applied

  • Scanlines: Every other pixel row at 5% black opacity
  • Chromatic aberration: Slight RGB channel offset at screen edges (2-3px)
  • Vignette: Radial gradient darkening corners
  • Screen curve: Very subtle barrel distortion

Implementation

  • Game renders to primary canvas as normal
  • Post-process pass draws to a second overlay canvas
  • Scanlines via repeating 2px gradient pattern
  • Chromatic aberration via 3x drawImage with globalCompositeOperation per channel

Toggle: 'C' key or button in pause/title screen. State saved to localStorage.

Count: 8-12 creatures, capped. Simple geometric shapes drawn with 3-5 canvas calls each.

AI: Lightweight boids — separation from each other, gentle drift, flee from ball (within 80px radius).

Reactivity: Cluster near active combos, swirl around power-ups, scatter to edges during boss fights.

Rendering: 10-15% opacity, drawn before the game layer. Never interfere with gameplay readability.

Audio sync: Creatures pulse slightly with the bass beat when procedural audio is active.
